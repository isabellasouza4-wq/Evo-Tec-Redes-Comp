# Evolução das Tecnologias de Redes de Computadores 
Este projeto é um site estático composto por três páginas interconectadas, desenvolvido como parte de uma atividade acadêmica de Web Design. O objetivo é apresentar a trajetória histórica das redes de computadores, desde a ARPANET até as tendências futuras como 5G e IoT.<br>
O projeto demonstra o uso prático de fundamentos do desenvolvimento Web, incluindo:<br>
HTML5: Estruturação de conteúdo com cabeçalhos, parágrafos e links.<br>
Elementos Semânticos: Uso de citações, listas ordenadas e não ordenadas, além de inserção de imagens.<br>
Navegação: Sistema de links internos entre os arquivos index.html, historia.html e futuro.html.<br>
